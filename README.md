# WP Visitor Stats

First-party **WordPress analytics** for logicencoder.com — page views, geography, technology breakdown, campaigns, custom events, IP bans, and a built-in URL shortener. Data stays in your database under operator control.

Private plugin: [logicencoder/wp-visitor-stats-plugin](https://github.com/logicencoder/wp-visitor-stats-plugin) (v1.4.x runtime).

## Tracking

JavaScript beacon on front-end pages (when enabled) plus optional server-side fallback on `wp`. Captures:

- Page URL, title, referrer, session ID
- Screen size, language, parsed browser/OS/device
- Geo country/region/city (via geo lookup)
- Bot vs human classification (`wp_visitor_stats_is_bot`)
- Scroll depth, page exit events, custom events with UTM campaign tags

Default retention: **180 days** with scheduled cleanup cron.

## URL shortener

Public redirects at **`{yoursite}/go/{slug}`** — create and manage links from admin with hit counts.

## IP bans

Banned IPs receive **403** before WordPress renders — useful for abuse patterns without editing server firewall rules.

## Admin dashboard

**Visitor Stats** menu (Chart.js + DataTables):

| Screen | Data |
|--------|------|
| Overview | Totals, trends, top pages |
| IPs / Live | Real-time and historical IP views |
| Geo | Map and country breakdown |
| Content | Top posts and landing pages |
| Technology | Browsers, OS, devices |
| Campaigns | UTM attribution |
| Custom Events | Named event counters |
| All Visitors | Searchable log |
| Ban List | Block/unblock IPs |
| URL Shortener | CRUD for `/go/` slugs |
| Diagnostics | Self-test, debug log tail |
| Settings | Enable/disable tracking, retention |

CSV export and database backup tools for operators.

## AJAX surface

Public (nopriv): `log_visitor_data`, `log_page_exit`, `log_scroll_depth`, `log_custom_event`.

Admin: 20+ actions for dashboard widgets, geo queries, ban management, short-link save/load.

## Integrations

Tracking snippet embeds on static HTML generated by [mexc-live-stats-plugin](https://github.com/logicencoder/mexc-live-stats-plugin-overview) snapshot pages — unified analytics across live and snapshot views.
